# How to Make Responsive Profile Card in HTML & CSS

Um projeto simples de **card de perfil responsivo** usando apenas HTML e CSS.

## ✨ Sobre

Este projeto mostra como criar um card de perfil moderno e adaptável para diferentes tamanhos de tela.

## 🛠️ Tecnologias

- HTML
- CSS

## 📱 Responsivo

Funciona em:
- Celular
- Tablet
- Desktop

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
